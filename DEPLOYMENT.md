# 📚 Deployment Guide - CAPTCHA Suite Web App\n\n## 🚀 GitHub Pages Deployment\n\n### Automatic Deployment\n\nYour CAPTCHA Suite is configured for automatic deployment via GitHub Pages.\n\n**Your Live Site:** `https://call-dot.github.io/captcha/`\n\n### Step-by-Step Setup\n\n1. **Go to Repository Settings**\n   - Navigate to: `Settings` → `Pages`\n   - URL: `https://github.com/Call-dot/captcha/settings/pages`\n\n2. **Configure Source**\n   - **Build and deployment** → **Source**\n   - Select: `Deploy from a branch`\n   - **Branch:** `main`\n   - **Folder:** `/ (root)`\n   - Click **Save**\n\n3. **Wait for Deployment**\n   - GitHub will build your site automatically\n   - First deployment takes 1-2 minutes\n   - Check the **Deployments** tab to see status\n\n4. **Access Your Site**\n   - Your site is now live at: `https://call-dot.github.io/captcha/`\n   - Share this URL with anyone!\n\n### ✅ What Gets Deployed\n\n```\ncaptcha/\n├── index.html                       ✅ Main landing page\n├── text-recognition-challenges.html ✅ Text challenges\n├── image-selection-challenges.html  ✅ Image challenges\n├── README.md                        ✅ Documentation\n├── package.json                     (Reference only)\n└── requirements.txt                 (Reference only)\n```\n\n---\n\n## 📝 Making Updates\n\n### Push Changes to Deploy\n\n```bash\n# 1. Make your changes\nvim index.html\n\n# 2. Stage changes\ngit add .\n\n# 3. Commit with message\ngit commit -m \"Update CAPTCHA challenges\"\n\n# 4. Push to main\ngit push origin main\n\n# 5. Wait 1-2 minutes\n# Your site will automatically update!\n```\n\n### Check Deployment Status\n\n1. Go to your repository\n2. Click **Deployments** tab\n3. See latest deployment status\n4. Click on deployment to see details\n\n---\n\n## 🔒 Security Settings\n\n### HTTPS Enforcement\n\n✅ **Already Enabled** for `call-dot.github.io` domain\n\n- All traffic is automatically redirected to HTTPS\n- Your site is secure by default\n- No configuration needed\n\n### Custom Domain (Optional)\n\nIf you want to use your own domain (e.g., `captcha.yoursite.com`):\n\n1.
